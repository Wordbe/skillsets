# vibe-slides

Zero-dependency HTML presentation generator for Claude Code.

## Features

- Single HTML file output — no npm, no build tools
- 12 distinctive style presets (Dark Botanical, Neon Cyber, Swiss Modern, etc.)
- PPT/PPTX conversion support
- Scroll-snap navigation with keyboard/touch/wheel support
- Responsive viewport fitting with animation

## Usage

```
/vibe-slides [topic or PPT file path]
```
